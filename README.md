# 🌌 Lumora Login

Uma interface de login moderna e responsiva desenvolvida utilizando apenas **HTML5** e **CSS3**.

O projeto apresenta um design inspirado em paisagens noturnas, utilizando tons de roxo, efeito glassmorphism e elementos visuais minimalistas para proporcionar uma experiência agradável ao usuário.


## ✨ Funcionalidades

- Tela de login moderna
- Design responsivo
- Efeito Glassmorphism
- Campos de usuário e senha
- Opção "Lembrar senha"
- Link para recuperação de senha
- Link para cadastro
- Layout desenvolvido apenas com HTML e CSS

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3

## 🎨 Conceitos Aplicados

- Flexbox
- Posicionamento de elementos
- Responsividade
- Glassmorphism
- Gradientes
- Camadas visuais (background com profundidade)


## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de HTML e CSS, explorando a criação de interfaces modernas e visualmente atraentes sem a utilização de JavaScript ou frameworks externos.

## 💜 Inspiração

O nome **Lumora** foi inspirado na união das palavras *Luz* e *Aurora*, representando um ambiente acolhedor e moderno através de uma identidade visual baseada em tons de roxo e paisagens noturnas.
